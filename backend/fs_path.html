üìÅ fs Module (File System)

What is fs module?

The fs (File System) module is used to work with files and folders in Node.js.

  • Create files
  • Read files
  • Write files
  • Update files
  • Delete files
  • Create / remove folders

  if (err) console.log(err);
  console.log(data);
});
üëâ Reads file asynchronously (non-blocking)

2Ô∏è‚É£ Write to a File

  if (err) console.log(err);
  else console.log("File written");
});

3Ô∏è‚É£ Append Data

  console.log("Data added");
});

4Ô∏è‚É£ Delete File

  console.log("File deleted");

üìÇ path Module

What is path module?

The path module is used to handle file and folder paths safely.

  • Windows uses \
  • Linux/Mac uses /
  • path makes code OS-independent

1Ô∏è‚É£ Join Paths (MOST USED)


const path = require("path");

const filePath = path.join(__dirname, "files", "data.txt");
console.log(filePath);

2Ô∏è‚É£ Get File Name


path.basename("/users/node/file.txt");
// file.txt

3Ô∏è‚É£ Get Directory Name


path.dirname("/users/node/file.txt");
// /users/node

4Ô∏è‚É£ Get File Extension


path.extname("file.txt");
// .txt

__dirname & __filename


console.log(__dirname);   // folder path
console.log(__filename);  // file path

üåê path Module ‚Äì Static File Server Example

Serving HTML file using path


const http = require("http");
const fs = require("fs");
const path = require("path");

const server = http.createServer((req, res) => {
  const filePath = path.join(__dirname, "public", "index.html");

  fs.readFile(filePath, (err, content) => {
    if (err) {
      res.writeHead(500);
      res.end("Error");
    } else {
      res.writeHead(200, { "Content-Type": "text/html" });
      res.end(content);
    }
  });
});

server.listen(3000);
üëâ Used in real projects to serve HTML, CSS, JS files
